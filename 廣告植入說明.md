# 廣告植入說明

全站已改為免費使用，抖內為隨喜。營收可依賴廣告。

## 廣告區塊位置

- **首頁 (index.html)**：`id="adIndex"` 的 `<div class="ad-slot">`，在「討論區」按鈕下方、分析結果上方。
- **儀表板 (dashboard.html)**：`id="adDashboard"` 的 `<div class="ad-slot">`，在避險燈號卡片下方。

## 如何貼上廣告

1. **Google AdSense**  
   申請通過後，在 AdSense 後台取得「廣告單元」的程式碼（通常是一段 `<script>` 或 `<ins>` + `<script>`）。  
   把整段程式碼貼進對應的 `ad-slot` div 裡，例如：

   ```html
   <div class="ad-slot" id="adIndex">
     <script async src="https://pagead2.googlesyndication.com/..."></script>
     <ins class="adsbygoogle" ...></ins>
     <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
   </div>
   ```

2. **其他廣告聯播網**  
   同樣把對方提供的 HTML/script 貼進 `ad-slot` 即可。

3. **若暫無廣告**  
   維持現狀即可，區塊會顯示為淺色虛線框，不影響使用。

## 注意

- GitHub Pages 為靜態託管，無法跑 PHP 等後端，廣告需用「前端嵌入」方式（script/iframe）。
- 若廣告商要求審核網站，可提供 GitHub Pages 網址與頁面截圖。
